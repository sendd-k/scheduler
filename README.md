# Interview Scheduler

Interview Scheduler is a Single Page Scheduling App using React and Axios. As the user you are able to switch between days (Monday - Friday). Create Interviews with a specific interviewer. Edit already scheduled interviews. Delete already scheduled interviews.

## Setup

Install dependencies with `npm install`.
Fork and clone https://github.com/lighthouse-labs/scheduler-api then follow the README file to access the API database.

## Running Webpack Development Server

```sh
npm start
```

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```
